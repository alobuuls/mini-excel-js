## 📖 Description

Aplicación inspirada en una hoja de cálculo desarrollada con **HTML, CSS y Vanilla JavaScript**.

Permite editar celdas, calcular fórmulas, seleccionar columnas y copiar valores, simulando funcionalidades básicas de Excel directamente en el navegador.

---

## ✨ Features

- 📦 Edición dinámica de celdas  
- ➗ Fórmulas tipo Excel  
- ⚡ Cálculo automático  
- 📋 Selección completa de columnas  
- ⌨️ Copiar datos usando `CTRL + C`  
- 🧹 Limpiar columnas usando `Backspace`  
- 🔄 Actualización reactiva del estado  
- 🧠 Gestión dinámica de datos en memoria
